# 🎨 Convertio - Image Converter

Simple and fast image converter to convert your images to different formats and download them in one ZIP file.

**Live Demo:** https://gr4v1x.github.io/convertio/

## Features

- ✅ Upload multiple images
- ✅ Convert to different formats (PNG, JPEG, WebP, BMP, GIF)
- ✅ Download all images in one ZIP file
- ✅ No server needed - everything works in your browser
- ✅ Beautiful and easy-to-use interface
- ✅ Drag & drop support
- ✅ Privacy first - no data sent anywhere

## How to Use

1. Visit the website
2. Drag images or click to select
3. Choose your desired format
4. Click "Convert and Download ZIP"
5. Download your converted images

## Requirements

- Modern browser (Chrome, Firefox, Safari, Edge)
- No installation needed

## Technologies Used

- HTML5 Canvas API for image conversion
- JSZip for creating ZIP files
- Vanilla JavaScript (ES6+)
- CSS3 Grid & Flexbox
